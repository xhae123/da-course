## 🟢 Part 1. Google Colab 처음 쓰는 사람을 위한 안내

이 코스의 노트북(`.ipynb` 파일)은 **Google Colab**에서 실행합니다.
Colab은 **내 컴퓨터에 아무것도 설치하지 않고**, 구글 계정만 있으면 브라우저에서 바로 Python 코드를 돌릴 수 있는 무료 서비스예요.

### 1) Colab이 뭔가요?
- 구글이 제공하는 **무료** 온라인 코딩 공간입니다.
- 코드를 **칸(셀) 단위**로 적고, 한 칸씩 실행하면서 결과를 바로 확인할 수 있어요.
- 글(설명) 칸과 코드 칸이 섞여 있어서, **교과서를 읽으면서 동시에 실습**하는 느낌입니다.

### 2) 준비물
- **구글 계정** (Gmail) 하나면 끝입니다.
- 크롬 브라우저를 권장해요.

### 3) 노트북 여는 방법 (가장 쉬운 길)

이 저장소의 노트북을 **클릭 한 번으로 Colab에서 바로 열 수 있습니다.** 아래 표의 "Colab에서 열기" 버튼을 누르세요.

> 💡 처음 열면 "경고: 이 노트북은 Google에서 작성하지 않았습니다"가 뜰 수 있어요. **[무시하고 실행]**을 누르면 됩니다. (이 노트북에는 위험한 코드가 없습니다.)

### 4) 코드 실행하는 법
- 코드 칸을 클릭한 뒤 **`Shift + Enter`** 를 누르거나, 칸 왼쪽의 **▶ (실행) 버튼**을 누릅니다.
- **반드시 위에서부터 순서대로** 실행하세요. 아래 칸이 위 칸의 결과를 사용하기 때문이에요.
- 처음 실행할 때 "런타임에 연결 중..."이 잠깐 뜨는데, 정상입니다. (구글이 무료 컴퓨터를 빌려주는 중)

### 5) 자주 막히는 부분 (FAQ)

| 증상 | 해결 |
|---|---|
| `NameError: name '...' is not defined` | 위쪽 칸을 실행 안 했어요. **맨 위부터 순서대로** 다시 실행하세요. |
| 그래프에서 **한글이 □□□** 로 깨짐 | 한글 폰트 설치 칸을 실행한 뒤, **[런타임] → [런타임 다시 시작]** 하고 처음부터 다시 실행하세요. |
| 한참 기다려도 안 끝남 | 칸 왼쪽이 ⏳(모래시계)면 실행 중입니다. 너무 오래면 **[런타임] → [런타임 다시 시작]**. |
| 내가 고친 게 저장되나요? | 원본은 안 바뀝니다. 내 구글 드라이브에 저장하려면 **[파일] → [Drive에 사본 저장]**을 누르세요. |
| 데이터 파일을 못 찾아요 (5편) | PISA 데이터를 먼저 다운받아 **내 구글 드라이브**에 올려야 합니다. (아래 5편 설명 참고) |

> ✏️ 노트북 안에서 `# ✏️ 여기에 코드를 작성하세요` 라고 적힌 칸은 **직접 채워보는 연습 문제**입니다.
> `___` (밑줄)을 알맞은 값으로 바꿔서 실행해 보세요.

## 🎓 Part 3. 가르치는 분(교사·강사)을 위한 활용 가이드

이 자료는 **수업 교재 + 실습 워크북**으로 함께 쓰도록 만들어졌습니다.

### 설계 의도
- **설치 장벽 0** — Colab만 쓰므로 컴퓨터실 환경 세팅에 시간을 뺏기지 않습니다.
- **읽기 → 실행 → 직접 채우기** 구조 — 각 노트북은 개념 설명(글 칸) → 예제 코드(코드 칸) → `✏️ 연습 문제` 순으로 반복됩니다.
- **하나의 일관된 주제(PISA·디지털 활용)** — 1편부터 5편까지 같은 맥락의 데이터를 다뤄서, 마지막에 실제 연구 보고서 한 편으로 이어집니다.

### 수업 운영 예시
- **회차당 1편**, 90~100분 (강의 40% + 실습 60%) 기준.
- 연습 문제(`✏️`)는 **그 자리에서 직접 풀게** 하고, 막히면 위 FAQ를 함께 보세요.
- 5편은 **모둠 프로젝트**로 진행하고, 노트북 마지막의 "보고서 요약 템플릿"을 결과물로 제출하게 할 수 있습니다.
- 5편 끝에는 **자체 설문으로 확장**(예: "AI 도구 활용과 학습 태도")하는 가이드가 있어, 후속 탐구 활동으로 연결됩니다.
